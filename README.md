# Scrapscallions
Scrapscallions is a game where players build the ultimate robot to battle in the arena and try to come out on top. It has a heavy emphasis on game AI and autonmous agents. It was created as a capstone project by Luke Rapier, Sam Costello, and Jacob Jeffcoat.
# Features
- Robust AI system built with a combination of GOAP and Steering Behaviors.
- A parts system that introduces new goals and actions to each agent.
- A strong user interface that strongly supports the intended user experience.
